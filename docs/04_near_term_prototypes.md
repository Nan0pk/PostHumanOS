# Near-Term Prototype Wedges

The first engineering work should validate bounded mechanisms rather than attempt a full operating system.

## 1. Telemetry Scheduler

A scheduler that considers latency, joules, thermal headroom, accelerator availability, and model quota.

## 2. UCC Parser and Capability Registry

A parser and registry for declarative Universal Capability Contracts.

## 3. Semantic Storage Overlay

A graph/state overlay over existing storage that preserves compatibility while testing semantic persistence and provenance.

## 4. Sandboxed Probing Emulator

A safe emulator for testing bounded active probing and capability inference without risking real devices.

## 5. Restricted Verified Code DSL

A small language for machine-state-changing actions that can be deterministically checked before execution.

Each prototype must define success metrics, kill criteria, pivot rules, and safety assumptions.
