# Research Programs A–J

This directory modularizes the ten research programs defined in the foundation dossier and engineering roadmap.

| Program | File | Role |
|---|---|---|
| A | `A_intent_representation.md` | Formal intent and fulfillment semantics |
| B | `B_capability_contracts_telemetry.md` | Capability ontology, telemetry normalization, UCC |
| C | `C_safe_hardware_discovery.md` | Bounded discovery and safe probing |
| D | `D_multi_currency_orchestration.md` | Intent-based scheduling and resource control |
| E | `E_dynamic_morphology.md` | Dynamic topology and transition economics |
| F | `F_semantic_persistence.md` | Semantic state and storage replacement analysis |
| G | `G_certifiable_generation.md` | Structural compilation and verified generation |
| H | `H_deterministic_admission.md` | Trust, proof-bounded execution, admission |
| I | `I_environmental_networking.md` | Context-aware and adaptive networking |
| J | `J_governance_provenance.md` | Governance, provenance, compatibility |

Each program file should track objective, dependencies, near-term deliverables, evaluation criteria, risks, and related research artifacts.
