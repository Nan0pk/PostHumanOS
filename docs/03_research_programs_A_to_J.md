# Research Programs A–J

PostHumanOS is decomposed into ten assignable research programs.

| Program | Focus | First Deliverable |
|---|---|---|
| A | Intent representation and fulfillment semantics | Intent schema v0 |
| B | Capability ontology, telemetry, UCC | UCC schema v0 |
| C | Safe hardware discovery and bounded probing | Probe emulator |
| D | Multi-currency orchestration | Scheduler simulator |
| E | Dynamic morphology and transition economics | Reconfiguration cost model |
| F | Semantic-state persistence | Storage overlay |
| G | Structural compilation and certifiable generation | Restricted DSL |
| H | Trust, deterministic admission, proof-bounded execution | Admission policy v0 |
| I | Environmental embodiment and adaptive networking | Privacy-bounded context model |
| J | Governance, explainability, provenance, compatibility | Provenance graph schema |

See [`../programs/README.md`](../programs/README.md) for modular program files.
