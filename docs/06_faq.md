# FAQ

## Is PostHumanOS a real operating system today?

No. It is currently a research program and architectural decomposition.

## Is this just an AI agent controlling the OS?

No. AI may propose, classify, or assist, but machine-state-changing actions require deterministic admission.

## Does PostHumanOS claim files and source code are useless?

No. It treats them as historically successful abstractions with important governance functions. Any replacement must preserve or replace those functions.

## Why use thermodynamic language?

Because energy, heat, data movement, and physical resource limits increasingly shape system behavior. The repo requires these claims to map to measurable instrumentation.

## What should be built first?

Formal intent schemas, UCC schemas, multi-currency scheduling simulation, provenance graphs, and restricted verified DSLs.
