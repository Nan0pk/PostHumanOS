# Program F — Semantic-State Persistence

## Objective

Develop semantic state overlays that preserve provenance, rollback, and compatibility while reducing file centrality in bounded domains.

## Why This Matters

This program isolates one of the core bottlenecks needed before PostHumanOS can move from thesis to bounded experimental platform.

## Current Status

Research-defined; implementation artifacts pending.

## Key Research Questions

- What is the minimum formal structure needed for this program to support bounded prototypes?
- Which assumptions must be checked before integration with other programs?
- What failure modes would invalidate the current approach?
