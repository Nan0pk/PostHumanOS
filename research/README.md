## How to Extend This Directory

When adding research material:

- state which program(s) A–J it supports;
- cite relevant prior work or mark the claim as an open gap;
- identify whether each source is foundational, enabling, contrasting, or cautionary;
- preserve the architectural invariants used throughout the project.
